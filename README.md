# 📊 Retail & FMCG Sales Dashboard

**Tool:** Google Sheets + Looker Studio  
**Domain:** Retail & FMCG  
**Live Dashboard:** https://docs.google.com/spreadsheets/d/17QpBhuo8AW-iUp8DnR6KZ3Yc2_aVNOXYPQOsAFN6SGw/edit?usp=sharing

## Overview
Sales analysis of Retail & FMCG data across 5 regions, 
10 cities, 4 categories and 15 brands for 2023–24.

## Key Metrics
- Total Revenue: ₹17.8L+
- Top Category: Personal Care
- Top City: Mumbai  
- Top Channel: Modern Trade

## Tools Used
- Google Sheets — data storage
- Looker Studio — visualization & dashboard

## Key Insights
1. Personal Care drives highest revenue
2. Mumbai + Delhi = 45% of total sales
3. E-Commerce fastest growing channel
4. Q4 festival season spikes visible in trend
